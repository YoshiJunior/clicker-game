# Biscuit Bazaar

A self-contained clicker game inspired by Cookie Clicker. It runs as a static site, so you can host it on GitHub Pages with no build step.

## Run locally

Open `index.html` in a browser, or use any static file server.

## GitHub Pages

1. Push these files to a GitHub repository.
2. In the repository settings, open `Pages`.
3. Set the source to the branch that contains this folder and publish from the root.
4. Wait for the Pages URL to become available.

## What it includes

- Big clickable biscuit
- Passive income buildings
- Scaling shop prices
- Autosave with `localStorage`
- Offline progress when you return later
